Présentation

Notre application web vous permet de réserver un vélo à Lyon en quelques clics !
Consultez le diaporama ci-dessous afin d'obtenir plus d'informations sur notre service.

  • Sélectionnez une station pour afficher ses informations.

    Si l'icône est rouge, la station est fermée.

    Si l'icône est verte, réservez votre vélo !

  • Pour réserver un vélo,

    signez dans le champ prévu à cet effet.

    Cliquez ensuite sur le bouton "Signer" et le tour est joué !

  • Votre réservation s'affichera dans le pied de page.

    Elle expire automatiquement au bout de 20 minutes.
